# GitHub Source Links

**Structure**: `https://github.com/{org}/{repo}/blob/{commit}/{path}#L{start}-L{end}`

| Type | Format |
|------|--------|
| Single line | `#L42` |
| Line range | `#L36-L80` |

**Tip**: Press `y` on any file view to convert branch URL to commit permalink.

> [GitHub Docs](https://docs.github.com/en/repositories/working-with-files/using-files/getting-permanent-links-to-files)
