# GitLab Source Links

**Structure**: `https://gitlab.com/{group}/{repo}/-/blob/{commit}/{path}#L{start}-{end}`

| Type | Format |
|------|--------|
| Single line | `#L42` |
| Line range | `#L36-80` |

Note: `/-/blob/` prefix. Line ranges have no second `L`.

**Tip**: Press `y` to convert to permalink.

> [GitLab permalinks](https://makandracards.com/makandra/31613-getting-permanent-links-to-files-on-github-or-gitlab)
